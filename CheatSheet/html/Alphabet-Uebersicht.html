Alphabetische Übersicht von Befehlen

  • git --version
    • Zeigt an, welche Version von Git installiert ist bzw. führt zur Installation
  • git add .
    • Hinzufügen aller Files zum ausgewählten Projekt für den Upload
  • git branch -M main
    • Definiert die Branch des Uploads
  • git commit -m "Name für den Upload vergeben"
    • Vergabe eines Namens für den aktuellen Upload - im Falle von Aktualisierungen eines bereits existierenden Projekts ggf. angeben, was angepasst wurde
  • git config --global --list
    • Mit diesem Befehl können die zuvor angegebenen Daten noch einmal abgerufen und geprüft werden
  • git config --global user.email
    • Hier wird als "e-mail address" die mit dem Github Profil verbundene E-Mail-Adresse angegeben
  • git config --global user.name
    • Hier wird als "username" der jew. Username des zu verbindenden Github Profils angegeben
  • git init
    • Zur Initialisierung der Übertragung in ein bestehendes Repository
  • git push -u origin main
    • Überträgt schließlich die lokalen Daten in das Repository auf Github
  • git remote add origin https://github.com/"username"/"Name Repository".git
    • Fügt ein Remote-Repository hinzu
